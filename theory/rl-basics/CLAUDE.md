기초 강화학습 알고리즘과 강화학습 환경을 만들어 해당 알고리즘 및 코드를 테스트 할 수 있는 블로그를 완벽하게 만들고 싶습니다.
기초 강화학습은 이론과 코드 모두 완벽하게 커버할 수 있어야 하며, 저와 해당 블로그를 보는 사람들 모두 이해하기 쉽게 작성되어야 합니다.

각 주차에 해당하는 기초 강화학습 이론과 코드는 아래의 내용을 포함해주어야 합니다:
- 자료는 반드시 영어로 만들어져야 합니다. (영어 버전과 한국어 버전을 둘 다 볼 수 있도록 버튼을 만들어주세요.)
- 각 방법론에 대한 수학적 증명
- 코드는 반드시 JAX를 이용하여 만들어져야 함.
- 코드와 수식이 해당 어디에 해당하는 내용인지 이해하기 쉽게 설명이 되어 있어야 합니다.
- figure가 필요한 부분에는 figure가 삽입 될 수 있도록 해야합니다.
- 또한, Week마다 Materials를 클릭하면 설명하는 블로그 페이지로 이동해야 합니다.
- Youtube 링크는 나중에 강의를 찍어 업로드할 예정입니다.

각각의 주차도 html파일로 완벽하게 만들어주세요. 각각의 주차마다 폴더를 만들어서 해당 페이지로 이동할 수 있도록 분리해주세요.
